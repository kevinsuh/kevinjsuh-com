# kevinjsuh.com

Personal website for Kevin Suh - Builder, Investigator, Craftsman.

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Development

```bash
npm run dev    # Start dev server
npm run build  # Build for production
npm run start  # Start production server
```

## Structure

- `/` - Homepage with about and links
- `/blog` - Blog posts (coming soon)

Built with intention. Every line deliberate.
